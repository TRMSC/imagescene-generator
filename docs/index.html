Über das Projekt

Mithilfe des Imagescene Generators können Bilder in dynamische Szenen verwandelt werden. Beispielsweise können hierdurch Kurs- oder Themenbanner im Lernmanagementsystem Moodle mit dezenten Animationen erzeugt werden. Der erzeugte SVG-Code kann jedoch auch in anderen digitalen Szenarien eingesetzt werden.

Bildszene erstellen

Um eine Bildszene zu erstellen, wird eine lokale Datei oder die URL zu einer Bilddatei benötigt. Es kann außerdem ein Codeschnipsel verwendet werden, der die URL zu einem Bild beinhaltet. Beispielsweise kann bei der Arbeit in Moodle ein Bild in den dortigen Editor eingefügt und der gesamte, hierdurch erzeugte Code für den Imagescene Generator genutzt werden.

Der Vorteil bei der Verwendung eines Codeschnipsels ist, dass vorhandene Werte für das Seitenverhältnis sowie für den Alt-Tag nach Möglichkeit automatisch übernommen werden. ✨

Generator

Lokale Bilddatei als DataURI verarbeiten:
Mithilfe von DataURIs können Bilder in Text umgewandelt werden. Mit zunehmender Bildgröße kann die erzeugte Buchstabenfolge schnell umfangreich sowie unübersichtlich werden und es können zudem Probleme in der Darstellung und Verarbeitung auftreten. Es kann sich somit anbieten, anstelle der Umwandlung lokaler Dateien mit Bildadressen zu arbeiten.
Der Generator ruft die verwendete URL nicht auf, sondern verpackt diese zusammen mit dem animierten Overlay in einen SVG-Code.
Breite und Höhe beziehen sich auf die Maße des verwendeten Bildes. Entscheidend ist das Seitenverhältnis, sodass die Einheit nicht beachtet werden muss.
Die Verwendung einer alternativen Bildbeschreibung ist optional, wird jedoch aus Gründen der Barrierefreiheit empfohlen.
