Über das Projekt

Mithilfe des Imagescene Generators können Bilder in dynamische Szenen verwandelt werden. Beispielsweise können hierdurch Kurs- oder Themenbanner im Lernmanagementsystem Moodle mit dezenten Animationen erzeugt werden. Der erzeugte SVG-Code kann jedoch auch in anderen digitalen Szenarien eingesetzt werden.

Informationen zur Nutzung

Bildszene erstellen

Um eine Bildszene zu erstellen, wird die URL zu einer Bilddatei benötigt. Es kann dabei auch ein Codeschnipsel verwendet werden, der eine solche URL beinhaltet. Bei der Arbeit in Moodle kann im Editor ein Bild eingefügt werden und der gesamte, hierdurch erzeugte Code - oder nur die Bild-URL selbst - eingesetzt werden.

Hinweis: Der Generator selbst greift nicht auf die URL zu, sondern verpackt diese zusammen mit dem animierten Overlay in einen neuen SVG-Code.

Generator

* Breite und Höhe beziehen sich auf die Maße des Bildes. Entscheidend ist das Seitenverhältnis, sodass die Einheit nicht beachtet werden muss. Wenn die URL des Bildes als Code-Schnipsel mit den entsprechenden Abmessungen (z.B. aus dem Editor in Moodle heraus) eingefügt wird, werden die Werte automatisch ermittelt.
